7. Sample Answer for Trainer:
"How did you implement the AdminPayments page?"

"I created a React functional component that uses useState to manage the payments data, search inputs, and revenue. When the page loads, useEffect hook triggers API calls to the backend payment service through axios. The backend queries the MySQL database using JPA Repository and returns JSON data containing all payments and total revenue.

For the search functionality, when a user enters a Turf ID and clicks search, the frontend calls the getPaymentsByTurfId API which filters data at the database level using a WHERE clause. The filtered results update the component state, and React automatically re-renders the table with only matching payments.

The revenue box dynamically displays either total revenue for all turfs or filtered revenue for a specific turf, calculated by the backend using SQL SUM function on successful payments. All communication happens through our API Gateway which routes requests to the appropriate microservice.

The component follows React best practices with controlled components, proper lifecycle management using hooks, and separation of concerns by keeping API logic in a separate service file."â€‹
